# 🚀 빠른 시작 가이드

## 1️⃣ 설치

### 기본 설치 (간단 크롤러만)
```
run.bat 더블클릭
```

자동으로 필요한 패키지가 설치됩니다.

### 고급 크롤러도 사용하려면
```
setup_advanced.bat 더블클릭
```

약 5-10분 소요 (브라우저 다운로드 포함)

---

## 2️⃣ 실행

```
run.bat 더블클릭
```

---

## 3️⃣ 사용

### GUI가 나타나면:

```
┌─────────────────────────────────────────┐
│ 크롤러 선택:                            │
│ ● 간단 크롤러 (추천) ← 대부분 여기!     │
│ ○ 고급 크롤러                           │
│                                         │
│ URL: [________________________]         │
│                                         │
│ [크롤링 시작]                           │
└─────────────────────────────────────────┘
```

1. **크롤러 선택**
   - 간단 크롤러: Doxygen, Sphinx 등
   - 고급 크롤러: React, Vue 등 SPA

2. **URL 입력**
   - 예: https://developer.nvidia.com/.../index.html

3. **크롤링 시작** 클릭

---
